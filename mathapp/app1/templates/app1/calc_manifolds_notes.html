Norms and Inner Products

Let x and y be elements of an n-dimensional euclidean space and a be a real number. Then we define the norm of x as

|x| = \sqrt{\sum_{i=1}^{n} (x^{i})^{2}}

Theorem 1.1

(i) \ \ |x| \geq 0 \\ \\ \phantom{aaa} (ii) \ \ | \sum_{i=1}^{n} x^{i}y^{i} | \leq |x||y| \\ \\ \phantom{aaa} (iii) \ \|x + y| \leq |x| + |y| \\ \\ \phantom{aaa} (iv) \ \ |ax| = |a||x|

The inner product of x and y is defined as

\triangleleft x,y \triangleright = \sum_{i=1}^{n} x^{i}y^{i}

Theorem 1.2

(i) \ \ \triangleleft x,y \triangleright = \triangleleft y,x \triangleright \\ \\ \phantom{aaa} (ii) \ \ \triangleleft ax,y \triangleright = \triangleleft x,ay \triangleright = a \triangleleft x,y \triangleright \\ \\ \phantom{aaa} (iii) \ \ \triangleleft x_{1}+x_{2},y \triangleright = \triangleleft x_{1},y \triangleright + \triangleleft x_{2},y \triangleright \\ \\ \phantom{aaa} (iv) \ \ \triangleleft x,x \triangleright \geq 0 \\ \\ \phantom{aaa} (v) \ \ \ |x| = \sqrt{\triangleleft x,x \triangleright} \\ \\ \phantom{aaa} (vi) \ \ \ \triangleleft x,y \triangleright = \frac{|x+y|^{2} - |x-y|^{2}}{4}

Compact Sets

In the n dimensional real space, we define closed and open rectangles, respectively, by

[a_{1},b_{1}] \ \times \ ... \ \times \ [a_{n},b_{n}] \\ \phantom{aaa} (a_{1},b_{1}) \ \times \ ... \ \times \ (a_{n},b_{n})

Now we define U to be an open set if, given an open rectangle A, we have

U \ \subset \ \mathbb{R}^{n} \\ \phantom{aaa} \forall \ x \ \epsilon \ U \ \exists \ A: \\ \phantom{aaa} x \ \epsilon \ A \ \subset \ U

Conversely, a set C is closed if its compliment is open.

Given a set A that is a proper subset of the n dimensional real space, we define the following terms.

Interior of A: Given an open rectangle B

\exists \ B \ : \ x \ \epsilon \ B \ \subset \ A

You can imagine how given some set A, it would be possible to cover it using a union of open sets, where each element in A is conatined in at least one of the open sets in that cover.

A collection O of open sets is an open cover of A if for every element x in A, there exists an open set o in O such that x is in o. An example of an open cover of the real numbers is

O = \{(a,a+1) \ | \ a \ \epsilon \ \mathbb{R} \}

We now arrive at the definition of compactness.

A set A is called compact if for every open cover of A, say O, there is a finite subcollection of open sets in O which also covers A.

Ex. The empty set is compact. Any open cover that contains the empty set has a finite number of open sets that also contain the empty set.

The open interval (0,1) is not compact. Why? Well, to show this we only need find one open cover that does not have a finite number of open sets that does not cover (0,1). Consider the open cover defined by

\left(\frac{1}{n},1-\frac{1}{n}\right), \ \ n = 3,4,5,...

This completely covers (0,1), however it is clear that any finite subcollection will fall short of covering (0,1). Hence, (0,1) is not compact. We now come to an important theorem.

Theorem 1.3 (Heine-Barel): For a subset S of the n dimensional Euclidean space, the following statements are equivalent:

  1. S is closed and bounded
  2. S is compact
