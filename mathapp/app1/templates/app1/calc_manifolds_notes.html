Norms and Inner Products

Let x and y be elements of an n-dimensional euclidean space and a be a real number. Then we define the norm of x as

|x| = \sqrt{\sum_{i=1}^{n} (x^{i})^{2}}

Theorem 1.1

(i) \ \ |x| \geq 0 \\ \\ \phantom{aaa} (ii) \ \ | \sum_{i=1}^{n} x^{i}y^{i} | \leq |x||y| \\ \\ \phantom{aaa} (iii) \ \|x + y| \leq |x| + |y| \\ \\ \phantom{aaa} (iv) \ \ |ax| = |a||x|

The inner product of x and y is defined as

\triangleleft x,y \triangleright = \sum_{i=1}^{n} x^{i}y^{i}

Theorem 1.2

(i) \ \ \triangleleft x,y \triangleright = \triangleleft y,x \triangleright \\ \\ \phantom{aaa} (ii) \ \ \triangleleft ax,y \triangleright = \triangleleft x,ay \triangleright = a \triangleleft x,y \triangleright \\ \\ \phantom{aaa} (iii) \ \ \triangleleft x_{1}+x_{2},y \triangleright = \triangleleft x_{1},y \triangleright + \triangleleft x_{2},y \triangleright \\ \\ \phantom{aaa} (iv) \ \ \triangleleft x,x \triangleright \geq 0 \\ \\ \phantom{aaa} (v) \ \ \ |x| = \sqrt{\triangleleft x,x \triangleright} \\ \\ \phantom{aaa} (vi) \ \ \ \triangleleft x,y \triangleright = \frac{|x+y|^{2} - |x-y|^{2}}{4}

Compact Sets

In the n dimensional real space, we define closed and open rectangles, respectively, by

[a_{1},b_{1}] \ \times \ ... \ \times \ [a_{n},b_{n}] \\ \phantom{aaa} (a_{1},b_{1}) \ \times \ ... \ \times \ (a_{n},b_{n})

Now we define U to be an open set if, given an open rectangle A, we have

U \ \subset \ \mathbb{R}^{n} \\ \phantom{aaa} \forall \ x \ \epsilon \ U \ \exists \ A: \\ \phantom{aaa} x \ \epsilon \ A \ \subset \ U

Conversely, a set C is closed if its compliment is open.

Given a set A that is a proper subset of the n dimensional real space, we define the following terms.

Interior of A: Given an open rectangle B

\exists \ B \ : \ x \ \epsilon \ B \ \subset \ A

We can visualize the interior of A in the diagram below.

Exterior of A: Given an open rectangle B

\exists \ B \ : \ x \ \epsilon \ B \ \subset \ \overline{A}

We can visualize the exterior of A in the diagram below.

Boundary of A: Given an open rectangle B

\exists \ B \ : \ x \ \epsilon \ B \\ \phantom{aaa} B \ \cap \ A \neq \varnothing, \ \ B \ \cap \ \overline{A} \ \neq \ \varnothing

We can visualize the boundary of A in the diagram below.

You can imagine how given some set A, it would be possible to cover it using a union of open sets, where each element in A is conatined in at least one of the open sets in that cover.

A collection O of open sets is an open cover of A if for every element x in A, there exists an open set o in O such that x is in o. An example of an open cover of the real numbers is

O = \{(a,a+1) \ | \ a \ \epsilon \ \mathbb{R} \}

We now arrive at the definition of compactness.

A set A is called compact if for every open cover of A, say O, there is a finite subcollection of open sets in O which also covers A.

Ex. The empty set is compact. Any open cover that contains the empty set has a finite number of open sets that also contain the empty set.

The open interval (0,1) is not compact. Why? Well, to show this we only need find one open cover that does not have a finite number of open sets that does not cover (0,1). Consider the open cover defined by

\left(\frac{1}{n},1-\frac{1}{n}\right), \ \ n = 3,4,5,...

This completely covers (0,1), however it is clear that any finite subcollection will fall short of covering (0,1). Hence, (0,1) is not compact. We now come to an important theorem.

Theorem 1.3 (Heine-Barel): For a subset S of the n dimensional Euclidean space, the following statements are equivalent:

  1. S is closed and bounded
  2. S is compact

Ie - [a,b] is compact



Functions and Continuity

A function f is a mapping that takes each point in a domain A to a single point in the image f(A). We can write this component wise as follows:

f: \ A \ \longrightarrow \ \mathbb{R}^{m} \\ \phantom{aaa} f(x) = (f^{1}(x),f^{2}(x),...,f^{m}(x)), \ x \ \epsilon \ A

The following function is known as the ith projection:

\pi: \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R}^{m} \\ \phantom{aaa} \pi (x) = x, \ \pi^{i}(x) = x^{i} \\ \phantom{aaa} \pi^{i}: \ (x^{1},...x^{n}) \ \longrightarrow \ x^{i}

We define the limit of a function f as follows. For:

x \ \epsilon \ A

we say

\lim_{x \to a} f(x) = b

if

\forall \ \epsilon \ > \ 0, \ \exists \ \delta \ > \ 0 \\ \phantom{aaa} s.t. \ 0 \ < \ |x - a| \ < \delta, \ x \ \epsilon A

and we have

|f(x) - b| \ < \ \epsilon

We say that a function f is continuous at a point a if

\lim_{x \to a} f(x) = f(a)

Let's look at some simple examples to illustrate the concept of when a function is continuous at a given point and when it is not. First, the case where it is. Observe the following diagram:

It is clear that it is, because no matter how small we make the value e (that is, for any e > 0), we can always find a value d > 0 such that the definition of the limit holds. Now there is a lower limit of what e can be. So clearly, there will be an e > 0 such that we cannot find a d that satisies the criteria for continuity. We now come to the following theorems:

Theorem 1.4 Given a set A and a function f such that

f: \ A \ \longrightarrow \ \mathbb{R}^{m}

then f is continuous if and only if the following is satisfied:

\forall \ open \ set \ U \ \subset \ \mathbb{R}^{m} \\ \phantom{aaa} \exists \ open \ set \ V \ \subset \ \mathbb{R}^{n} \\ \phantom{aaa} s.t. \ f^{-1}(U) = V \ \cap \ A

Theorem 1.5 Given a set A and a function f such that

f: \ A \ \longrightarrow \ \mathbb{R}^{m}

where f is continuous and A is compact, then

f(A) \ \subset \ \mathbb{R}^{m}

is also compact.

We define the maximum of a function at a point a given some value delta as

M(a,f,\delta) = sup \{ f(x): \ x \ \epsilon \ A, \ |x-a| \ < \ \delta \}

We define the minimum of a function at a point a given some value delta as

m(a,f,\delta) = inf \{ f(x): \ x \ \epsilon \ A, \ |x-a| \ < \ \delta \}

We define the oscillation of a function at a point a by

\theta(a,f) = \lim_{\delta \to 0} | \ (M(a,f,\delta) - m(a,f,\delta) \ |

We now have the following theorems:

Theorem 1.6 A bounded function f is continuous if and only if

\theta(a,f) = 0

Theorem 1.7 Let A be a closed set in the n dimensional Euclidean space. If

f: \ A \ \longrightarrow \ \mathbb{R}

is any bounded function then for any epsilon greater than 0, the set

\{ x \ \epsilon \ A: \ \theta(x,f) \ \geq \ \epsilon \}

is closed.



Differentiation

A one variable definition of differentiation will not suffice for our studies. Instead, we need to generalize to any dimension. A function f from the n dimensional Euclidean space to the m dimensional Euclidean space is said to be differentiable at a point a if the following condition is met:

f: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R}^{m} \\ \phantom{aaa} \exists \ \lambda: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R}^{m} \\ \\ \phantom{aaa} \lim_{h \to 0} \frac{|f(a+h)-f(a)-\lambda(h)|}{|h|} = 0

Where lambda is a linear transformation, and is denoted by Df(a).

Theorem 1.8 Chain Rule

if \ f: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R}^{m} \\

is differentiable at a and

g: \ \mathbb{R}^{m} \ \longrightarrow \ \mathbb{R}^{p} \\

is differentiable at f(a), then

g\circ f: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R}^{p} \\

is differentiable at a and

D(g\circ f)(a) = Dg(f(a))\circ Df(a)

Corollary

(i) \ \ if \ f: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R}^{m} \ constant, \ \Df(a) = 0 \\ \phantom{aaa} (ii) \ if \ f: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R}^{m} \ linear, \ \ Df(a) = f \\

Corollary

If \ \ f,g: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R} \ \ C^{1} \ at \ a \\ \\ \phantom{aaa} (i) \ \ D(f +c g)(a) = Df(a) + Dg(a) \\ \\ \phantom{aaa} (ii) \ \ d(fg)(a) = g(a)Df(a) + f(a)Dg(a) \\ \\ \phantom{aaa} (iii) \ \ d(f/g)(a) = \frac{g(a)Df(a) - f(a)Dg(a)}{[g(a)]^{2}}


Partial Derivatives

We now turn our attention to derivatives of mulivariable functions. Of course, we know the definition from our first classes on multivariable calculus, but now we generalize the definition to arbitrary manifolds. If

f: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R} \\ \\ \phantom{aaa} D_{i}f(a) = \lim_{h \to 0} \frac{f(a^{1},...,a^{i}+h,...,a^{n})}{h}

exists, then we say the ith partial derivative of f exists. We can also define the second order (or in general, kth order) partial derivative by the same definition. In fact, we must have, for ith and jth components of the partial derivative, that

D_{ij}f(a) = D_{ji}f(a)

as long as the two derivatives are continuous at a. If f has continuous partials of all orders, we call f C infinite smooth. Other concepts carry over from our elementary calculus courses. For instance, if f maps some set A onto the reals, then we know that if f has a maximum (or minimum) at a point a on the interior of A and the ith partial of f exists at a, then this derivative must vanish at a. Note however, that the converse is generally not true (think of the function f(x)). We also have a chain rule of sorts for mixed partials, which is given in the following theorem:

Theorem 1.9

Let \ \ g_{1},...,g_{m} : \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R} \ be \ C^{\infty} \\ \\ \phantom{aaa} Let \ \ f: \ \mathbb{R}^{m} \ \longrightarrow \ \mathbb{R} \ \ be \ \ differentiable \ \ at \ \ (g_{1}(a),...,g_{m}(a)) \\ \\ \phantom{aaa} Let \ \ F: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R} \ \ be \ \ F(x) = f(g_{1}(x),...,g_{m}(x)) \\ \\ \phantom{aaa} Then \\ \phantom{aaa} D_{i}F(a) = \sum^{m}_{j=1}D_{j}f(g_{1}(a),...,g_{m}(a)) D_{i}g_{j}(a)


Inverse Functions

Now we recall the concept of an inverse function. The idea is straightforward enough: given a function f that maps a set A to a set B, the inverse of f is given by the mapping of B back onto A. The following theorem tells us about the relationship between the derivative of a function f and the derivative of its inverse.

Theorem 1.10 (Inverse Function Theorem)

f: \ \mathbb{R}^{n} \ \longrightarrow \ \mathbb{R}^{n} \ C^{1} \\ \\ \phantom{aaa} a \ \epsilon \ A \ open \\ \\ \phantom{aaa} det(D(f(a)) \neq 0 \\ \\ \phantom{aaa} Then \ \exists \ V \ open, \ a \ \epsilon \ V, \\ \\ \phantom{aaa} W \ open \ f(a) \ \epsilon \ W \\ \\ \phantom{aaa} s.t. \ f: \ V \ \longrightarrow \ C^{1} \ has \ inverse \\ \\ \phantom{aaa} f^{-1}: \ W \ \longrightarrow \ V \ satisfying \\ \\ \phantom{aaa} [D(f^{-1}(y)]^{-1} = Dff^{-1}(y)


Integration

Recall that a partition p of [a,b] is a sequence t0, t1, ... , tn that splits [a,b] into segments (not neccessairly of even length). We can visualize this in the diagram below.

Here, t0 = a and tn = b. Now we would like to abstract this concept to higher dimensions, so instead of talking about the partition of an interval, we talk about the partition of an n dimensional rectangle. So we say a partition of a rectangle [a1,b1] x [a2,b2] x ... x [an,bn] is a collection p = (p1, p2, ... ,pn) where pi is a partition of [ai,bi]. If pi divides [ai,bi] into Ni subrectangles, then p divides [a1,b1] x [a2,b2] x ... x [an,bn] into N1, N2, ... , Nn subrectangles. Now suppose we are looking at a function over the domain of these subrectangles, say f. Then for each subrectangle S, we have

m(f) = inf\{f(x): \ x \ \epsilon \ S\} \\ \\ \phantom{aaa} M(f) = sup\{f(x): \ x \ \epsilon \ S \}

Each of the subrectangles S of the partition p has a capacity. In one dimension it is more commonly known as the length, in two dimensions the area, and in three dimensions the volume. We will call the capacity the volume, regardless of what dimension we are working in (This can be confusing at times, but it is the convention used in this course, so keep it in mind). Then let V(S) represent the volume of a particular subrectangle S for the partition p. The lower and upper sum of f for p are

L(f,p) = \sum_{S}m_{S}(f)V(S) \\ \\ \phantom{aaa} U(f,p) = \sum_{S}M_{S}(f)V(S)

For instance, suppose we were dealing with the one dimensional case where we partition [a,b] into n segments. The image below shows the maximum sum U(f,p) given the partition p and the function f over [a,b]:

Now suppose we had some partition p of a rectangle [a1,b1] x [a2,b2] x ... x [an,bn]. It is intuitively obvious that we can always find another partition, say p', that more accurately represents the lower and upper sum of f over p. We can do this by cutting the rectangles into smaller segements. This new partition p' is called a refinement of p.

Lemma: Suppose we have some function f and some partition p of a rectangle [a1,b1] x [a2,b2] x ... x [an,bn]. Then if p' refines p, we have

L(f,p) \leq L(f,p') \\ \\ \phantom{aaa} U(f,p') \leq U(f,p)

The reason for this lemma should be quite obvious. If we refine a lower sum, then we expect to make the rectangles S to rise to the level of the function, increasing the sum. On the other hand, if we refine an upper sum, then we expect to make the rectangles S lower to the level of the function, decreasing the sum.

Corollary: If p and p' are two partitions, then

L(f,p') \leq U(f,p)

A function f mapping a rectangle A to the real numbers is called integrable if f is bounded and sup{ L(f,p) } = inf{ U(f,p) }. In this case, we denote the integral of f over A as

\int_{A} f \ dx = \int_{A} f(x^{1},...,x^{n})dx^{1}...dx^{n}

Theorem 2.0: A bounded function f is integrable on a rectangle A iff:

\forall \ \epsilon \ > \ 0 \ \ \exists \ \p \ s.t. \ U(f,p) - L(f,p) \ < \ \epsilon


Measure Zero and Content Zero

A set A has measure zero if for any epsilon greater than zero, there exists a cover {U1, U2, ...} of closed rectangles such that

\sum_{i=1}^{\infty} V(U_{i}) \ < \ \epsilon

The union of sets all of measure zero results in a set that has measure zero.

A set A has content zero if for any epsilon greater than zero, there exists and finite cover {U1,...Un} of closed rectangles such that

\sum_{i=1}^{n} V(U_{i}) \ < \ \epsilon

Theorem 2.1: If A is compact and has measure zero, then A has content zero.

Theorem 2.2:

A \ closed \ rectangle \\ \\ \phantom{aaa} f: \ A: \ \longrightarrow \ \mathbb{R} \ bounded \\ \\ \phantom{aaa} B = \{x: \ x \ \epsilon \ \A, \ f != C \ at \ x \} \\ \\ \phantom{aaa} Then \ f \ is \ integrable \ iff \ B \ has \ measure \ zero

We define the characteristic function as follows:

\forall \ C \ \subset \ \mathbb{R}^{n} \\ \\ \phantom{aaa} \mathcal{X}_{c}(x) = \begin{cases} 1,& \text{if } \ x \notin C \\ 0, & x \in C \end{cases}

Using this characteristic function, we can now recast our integerals to be computed over any arbitrary surface, not just rectangles. Let C be a subset of A, where A is a closed rectangle over the n dimensional Euclidean space. Then we define:

\int_{C} f = \int_{A} f \mathcal{X}_{C}

Provided both f and X are integrable.

Theorem 2.3: The characteristic function X: A => R is integrable iff the boundary dC has measure zero.

A bounded set C whose boundary dC has measure zero is called Jordan Measureable.



Fubini's Theorem

Now that we have defined integration in a sensible way, we turn our attention to the interchangeability of multiple integration of functions.

Fubini's Theorem: Suppose A and B are closed rectangles and subsets of the n-dimensional Euclidean space, and f is a function that takes A x B to R and is integrable.

\text{for} \ x \ \epsilon \ A, \ \text{let} \ g_{x} = f(x,y) \\ \\ \phantom{aaa} \text{let} \ \mathcal{L}(x) = L\int_{B}g_{x} = L\int_{B}f(x,y)dy \\ \\ \phantom{aaa} \mathcal{U}(x) = U\int_{B}g_{x} = U\int_{B}f(x,y)dy \\ \\ \phantom{aaa} \text{then} \ mathcal{L} \ \text{and} \ \mathcal{U} \ \text{are integrable on A, and} \\ \\ \phantom{aaa} \int_{AxB}f = \int_{A}\mathcal{L} = \int_{A} \left( L\int_{B}f(x,y)dy \right) dx \\ \\ \phantom{aaa} \int_{AxB}f = \int_{A}\mathcal{U} = \int_{A} \left( U\int_{B}f(x,y)dy \right) dx \\ \\


Partitions of Unity

Theorem 2.4

\text{Let} \ A \ \subset \ \mathbb{R}^{n} \ \text{and} \ \theta \ \text{be an open cover of A} \\ \\ \phantom{aaa} \text{then} \ \exists \ \text{a collection} \ \Phi \ \epsilon \ C^{\infty} \ \text{functions} \ \phi \ \text{defined in an open set containing A} \\ \\ \phantom{aaa} s.t. \\ \\ \phantom{aaa}(i) \ \forall \ x \ \epsilon \ A, \ 0 \ \leq \ \phi(x) \ \leq \ 1 \\ \\ \phantom{aaa}(ii) \ \forall \ x \ \epsilon \ A, \ \exists \ \text{an open set V containing x such that all but finitely many} \\ \phantom{aaa} \phi \ \epsilon \ \Phi = 0 \ \text{on V} \\ \\ \phantom{aaa}(iii) \ \forall \ x \ \epsilon \ A, \sum_{\phi \epsilon \Phi}\phi(x) = 1 \\ \\ \phantom{aaa}(iv) \ \forall \ \phi \ \epsilon \ \Phi, \ \exists \ \text{an open set} \ U \ \epsilon \ \theta \ s.t. \\ \phantom{aaa} \phi = 0 \ \text{outside some closed set contained in U}

Remark: A collection of functions satisfying conditions (i) through (iii) is called a partition of unity, while a collection of functions satisfying conditions (i) through (iv) is called a partition of unity subordinate to the open cover theta.

An open cover theta of an open set A is said to be admissible if:

\forall \ U \ \epsilon \ \theta, \ U \ \subset \ A
\text{If} \ \Phi \ \text{is subordinate to} \ \theta, \ \text{and} \ f: \ A \ \longrightarrow \ \mathbb{R} \ \text{is bounded, and} \\ \\ \phantom{aaa} \{x : \ f \ \text{discontinuous at x} \} \ \text{has measure 0, then} \\ \\ \phantom{aaa} \int_{A}\phi|f| \ \text{exists} \ \forall \ \phi \ \epsilon \ \Phi \ \text{a partition of unity}

We define f to be integrable in the extended ense if

\sum_{\phi\epsilon\Phi} \ \int_{A}\phi \ f \ \text{converges, in which case we denote this integral as} \ \int_{A}f

Theorem 2.5

(i) If psi is another partition of unity (from our already defined phi) subordinate to an abmissable cover theta prime of A, then

\sum_{\psi\epsilon\Psi} \int_{A} \psi |f| \ \text{also converges, and} \\ \\ \phantom{aaa} \sum_{\phi\epsilon\Phi} \int_{A}\phi \ f = \sum_{\psi\epsilon\Psi} \int_{A} \psi \ f

IE - It doesn't matter what partition of unity you take to integrate.

(ii) If A and f are bounded, then f is integrable in the extended sense.

(iii) If A is Jordan measureable and f is bounded, then this definition agrees with the old one.

Tensors

Let V be a vector space. Denote:

V \times ... \times V = V^{k} \\ \\ \phantom{aaa}T: \ V^{K} \ \longrightarrow \ \mathbb{R}

T is called a k-tensor if the following conditions are met:

\forany \ 1 \ \leq \ i \ \leq \ k, \ a \ \epsilon \ \mathbb{R}: \\ \\ \phantom{aaa} T(v_{1},...v_{i}+v+{i}',...,v_{k}) = T(v_{1},...v_{i},...,v_{k}) + T(v_{1},...v+{i}',...,v_{k}) \\ \\ \phantom{aaa} T(v_{1},...a*v_{i},...,v_{k}) = a*T(v_{1},...v_{i},...,v_{k})

The set of all k-tensors, denoted by:

\tau^{k}(v)

is a vector space in of itself. Consider any two tensor S and T in this vector space, and some scalar a. We have the following properties:

(S + T)(v_{1},...,v_{k}) = S(v_{1},...,v_{k}) + T(v_{1},...,v_{k}) \\ \\ \phantom{aaa} (aS)(v_{1},...,v_{k}) = a*S(v_{1},...,v_{k})

We can also define the tensor product of two tensors S and T. It is given by:

S \ \epsilon \ \tau^{k}(v), \ v \ \epsilon \ V^{k} \\ \\ \phantom{aaa} T \ \epsilon \ \tau^{l}(v), \ v \ \epsilon \ V^{l} \\ \\ \phantom{aaa} S\otimes T \ \epsilon \ \tau^{k+l}(v) \\ \\ \phantom{aaa} (S\otimes T)(v_{1},...v_{k+l}) = S(v_{1},...v_{k}) T(v_{1},...v_{l})

The tensor product has the following properties:

(S_{1}+S_{2})\otimes T = S_{1}\otimes T + S_{2}\otimes T \\ \\ \phantom{aaa} S \otimes (T_{1}+T_{2}) = S\otimes T_{1} + S\otimes T_{2} \\ \\ \phantom{aaa} (aS)\otimes T = a(S\otimes T) = S \otimes (aT) \\ \\ \phantom{aaa} (S\otimes T) \otimes U = S \otimes (T\otimes U)

Here it is worth noting that tensor products are generally not commutative.
