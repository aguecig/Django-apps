Calculus on Manifolds: Problem Set

Question 1: The quantity |y - x| is called the distance between x and y. Prove and interpret geometrically the "triangle inequality":

|z - x| \leq |z - y| + |y - x|


Question 2:

a) Given a closed set A and an element x such that x does not belong to A, prove there exists a number d with the following property:

d > 0 \ : \ |y - x| \geq d \ \forall \ y \ \epsilon \ A

b) Given A is a closed set, B is a compact set and

A \ \cap \ B = \varnothing

Prove that

\exists \ d > 0 \ : \ |y - x| \ \geq \ d \ \forall \ y \ \epsilon \ A, \ \forall \ x \ \epsilon \ B

c) Give a counter example in R2 of A and B are closed but neither is compact.
