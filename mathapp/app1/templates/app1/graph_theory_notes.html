History of Graph Theory

Consider the 18th century problem of the seven bridges of Konigsberg. In one of the areas of the city lie a network of rivers and bridges, as seen in the diagram below.

The problem is outlined as follows: Is there a way for a person to travel along each bridge, but only exactly once? In the year 1736, Euler was able to prove that the answer is a definitive no. Given a few trial runs, we may be able to convince ourselves that Euler's conclusion is indeed correct. Only by developing a new form of mathematical analysis, graph theory, was Euler able to show with absolute certainty that indeed there is no solution to this problem, as well as solving a whole host of related problems.

Terminology of Graphs

Vertices are points on a graph that represent some kind of data. These vertices can be connected by edges. Faces are parts of the plane that are enclosed by edges. If we can draw these vertices and edges such that no edges intersect each other, the graph is called a planar graph.



In the example shown above, the vertices are A, B, C and D. The edges are AB, AC, BC and CD as they connect these vertices. There are two faces: the one enclosed by ABC, and the rest of the plane. The graph above is planar because the edges do not intersect eachother.

The degree of a vertex is equivalent to the number of edges that extrude from it. In the example above, the degree of vertex C is 3, while the degree of vertex D is 1. The order of the graph is just the number of vertices it contains. The above example has order 4. Two vertices are said to be adjacent if there is an edge that connects them. Graphs and their Degree Sequences

We begin this section with the following theorem. Let G = (V,E) be a finite simple graph. Then

\sum_{v \epsilon V} deg(v) = 2|E|

This is known as the degree sum theorem. In words: the sum of the degrees of all vertices is twice the number of edges. The proof for this is rather trivial: every edge must be connected to two and only two vertices. QED

A degree sequence of G = (V,E) is the sequence of degrees of vertices of G in decreasing order. A sequence of non-negative integers in decreasing order is graphic if it is the degree sequence of some graph. Let's look at some examples.

Ex. The degree sequence given by 1, 1, 1 is not graphic because it cannot represent a graph. In this case, the graph would have to contain three vertices, and each vertice would have to be adjacent to exactly one other vertice. This is impossible.

Ex. The degree sequence given by 1, 1, 1, 1 is graphic. See the diagram below for an example of a graph with this degree sequence.

We now come to the hand shaking lemma. Let G = (V,E) be a finite simple graph. Then the number of vertices in G with odd degree must be an even amount. Stated otherwise:

V = (V_{even}) U (V_{odd}) \\ \\ \phantom{aaa} V_{odd} \ mod(2) = 0

So, for example, we know that the degree sequence given by 5, 4, 3, 2, 2, 2, 1 is not graphic. A degree sequence like 4, 3, 2, 2, 2, 1, 1, 1, 0 could be graphic, but we do not know from the hand shaking lemma. We simply cannot rule out the possibility that it is graphic from the lemma. A better tool for this is theorem by Hakimi, which we now state.

Consider sequences given by

(1) s_{1}, t_{1}, ... , t_{s}, d_{1},...,d_{n} \\ \\ \phantom{aaa} (2) t_{1}-1, ... , t_{s}-1, d_{1},...,d_{n}

Where both sequences are decreasing. Then (1) is graphic if and only if (2) is graphic. Take the following sequence: 6, 6, 6, 6, 4, 3, 3, 0. We begin by observing that we cannot dismiss this sequence as not graphical from the hand shakeing lemma. However, we can see that the sequence takes the form of (1) in the above theorem. Applying the theorem, we get

6, 6, 6, 6, 4, 3, 3, 0 \\ \phantom{aaa} \ \ \ 5, 5, 5, 3, 2, 2, 0 \\ \phantom{aaa} \ \ \ \ \ \4, 4, 2, 1, 1, 0 \\ \phantom{aaa} \ \ \ \ \ \ \ \ \ 3, 1, 0, 0, 0 \\

The last sequence is clearly not graphic, as show below:

So then we know the original sequence cannot be graphic either.

Though these two graphs have the same number of degree and edges, they differ in degree sequence. The first graph has a degree sequence of 3, 1, 1, 1 while the second graph has a degree sequence of 2, 2, 1, 1. These two graphs are structurally different. This leads us to the following definition.

Two finite simple graphs

G_{1} = (V_{1},E_{1}) \ \ \ G_{2} = (V_{2},E_{2})

are said to be isomorphic if there exist a bijection

f: \ V_{1} \ \longrightarrow \ V_{2}

that satisfies the following condition: For and u and v in V1 that are adjacent in G1, f(u) and f(v) are in V2 and are adjacent in G2.

The diagram below shows all graphs of degree 3 up to isomorphism.

There are certain structural properties of a graph that will remain unchanged under isomorphism. Such a concept is known as isomorphic invariance of a graph. There are many examples of these, and we will discuss a few now.

The order of a graph G = (V,E) : |V| is one such isomorphic invariant. In other words, if we have to graphs that are isomorphic, then they must contain the same number of vertices. This is a trivial result.

The degree sequence of a graph must be isomorphic invariant, as must the sum of all degrees of a graphs vertices. From this it also follows that the number of edges of a graph is an isomorphic invariant.



Graph Classes

Certain types of graphs are important and frequent enough that they deserve their own classification. Here is a selection of some of these graphs.

Path of length n

P_{n} = (V = \{0,1,...,n\}, \ E = \{(i,i+1) \ | \ i = 0,...n-1 \})

Cycle of length n

C_{n} = (V = \mathbb{Z} / \mathbb{Z}_{n}, \ E = \{(i,i+1) \ | \ i \ \epsilon \ \mathbb{Z} / \mathbb{Z}_{n} \} )

Complete graph on n vertices

K_{n} = (V = \{(1,...,n\}, \ E = [V]^{2} )

In other words, all vertices are connected to each other.

Complete bi-partite graph on n and m vertices

K_{n,m} = (V = V_{1} \ U \ V_{2}, \ E = \{(v_{1},v_{2}) \ | \ v_{1} \ \epsilon \ V_{1} \ \ v_{2} \ \epsilon \ V_{2} ) \\ \phantom{aaa} |V_{1}| = n, \ |V_{2}| = m

In other words, every vertex in V1 connects with a vertex in V2, however no vertices in V1 are adjacent with other vertices in V1, and the same is true of V2.
