Definitions

The limit of a function f(x) is the value of f(x) as x approaches some number c. This is written as follows:

\lim_{x\to c} f(x)

This is read as "The limit of f(x) as x approaches c."

The limit is said to exist if the value of f(x) as x approaches c from the left hand side is equivalent to the value of f(x) as x approaches c from the right hand side. We write this as

\lim_{x\to c^{-}} f(x) = \lim_{x\to c^{+}} f(x)

Suppose these limits are equal to some number L. Then the limit is written as

\lim_{x\to c} f(x) = L

If the left hand side limit does not equal the right hand side limit, then the limit does not exist:

\lim_{x\to c^{-}} f(x) \neq \lim_{x\to c^{+}} f(x) \\ \\ \phantom{aaa} \lim_{x\to c} f(x) = DNE

A function f(x) is said to be continuous at a point x = c if the following two conditions are met:

  1. The limit exists at x = c
  2. The value of the function at x = c is the same value as the limit at x = c

Mathematically, these two conditions can be written as

\lim_{x\to c^{-}} f(x) = \lim_{x\to c^{+}} f(x) = L \\ \\ \phantom{aaa} f(c) = L

If these conditions are not meant, f(x) is said to be discontinuous at x = c.

Techniques for Limit Evaluation

The first thing one should do is simply evaluate the function f(x) at the limit point c. If the function is one that is obviously continuous everywhere (such as a linear equation), then the limit value will always just be equal to the function value. Consider the following example:

\lim_{x\to 4} 3x - 4

This function is quite obviously going to be continuous everywhere, so we know that as we approach 4 from both the left hand side and the right hand side, we will be approaching the exact value of the function at x = 4.

\lim_{x\to 4} 3x - 4 = 3(4)-4 = 8

If however, we have division by 0, then we are dealing with a discontinuity (the function is not continuous at the point in question since f(c) does not even exist). If we have something of the form a/0, where a is non-zero, then we simply conclude that the limit does not exist. For instance:

\lim_{x\to 1} \frac{2}{x-1} \\ \\ \phantom{aaa} \frac{2}{1-1} = \frac{2}{0}

Therefore we can conclude that the left hand side limit does not equal the right hand side limit, so the limit does not exist. Indeed, if you recall your rational functions studies from your other math class, you will recognize that the funtion in the above example has the following graphical representation

From this graph we can clearly see that as x approaches 1 from the left, the value of the function approaches negative infinity. However as x approaches 1 from the right, the value of the function approaches positive infinity. Thus, the limit does not exist.

If we come across something of the form 0/0, we can be assured that the limit does in fact exist. This is because the discontinuity resulting from 0/0 represents a hole, and you can imagine how a hole will still have the same left hand and right hand limits. Take the following example:

\lim_{x\to 3} \frac{x^{2}-9}{x-3} \\ \\ \phantom{aaa} \frac{3^{2}-9}{3-3} = \frac{0}{0}

To find limits of the form 0/0, we can try the following strategies:

  • Factoring
  • Radical conjugation
  • Common denominators

In the above example, we can factor and simplify to the point where we are no longer dividing by 0 and then we can evaluate the limit directly:

\lim_{x\to 3} \frac{x^{2}-9}{x-3} \\ \\ \phantom{aaa} =\lim_{x\to 3} \frac{(x-3)(x+3)}{x-3} \\ \\ \phantom{aaa} =\lim_{x\to 3} x+3 \\ \\ \phantom{aaa} = 6
