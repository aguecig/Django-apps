Question 1: Determine the value of the following limit

\lim_{x\to 3} \frac{45 - 5x^{2}}{8x^{2} - 14x -30}

Hint: Try factoring first

Question 2: For which values of a does the following limit not exist?

\lim_{x\to a} \frac{\sqrt{x+9}-3}{x}

Hint: Remember that for real numbers, a square root of a negative number does not exist

Question 3: What value of k makes the following function continuous for all real numbers?

f(x) = \left\{ \begin{array}{ll} 4 - 7x & x\leq -3 \\ x^{2} - k & x > -3 \\ \end{array} \right.

Hint: For a piecewise function to be continuous, it must not have any jumps or holes where the pieces are seperated

Question 4: For the function given by

L(x) = \left\{ \begin{array}{ll} x+2 & x < 4 \\ \sqrt{x} & x > 4 \\ -11 & x = 4 \end{array} \right.

Determine the value of

\lim_{x\to 4^{+}} L(x)

Hint: This is the limit as we approach x = 4 from the right hand side of the function

Question 5: For the function given by

Determine the value of

s(7)

Hint: This is the actual value of the function at x = 7
