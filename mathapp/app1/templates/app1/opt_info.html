What is optimization?

Often it is the case when we are trying to solve some sort of problem, be it mathematical or otherwise, that there tends to be multiple solutions. Optimization is the process in which one tries to find the best possible (most efficient) solution to a given problem. A typical optimization problem will contain the following:

  • A set of constraints, or the given quantities of the problem
  • An unknown quantity that is to be either maximized or minimized (ie optimized)

An Example

Take the following example: given 240 cm of string, you are to design a rectangle. What are the dimensions of the rectangle that has the largest possible area? Let us list out our constraint and the quantity to be optimized:

  • Constraint: 240 cm of string, which will be used to construct the perimeter of the rectangle
  • Optimize: The area of the rectangle, which is to be maximized

We begin by defining our variables. Let L represent the length of the rectangle, W represent the width and A represent the area. Now we write out the two formulas that represent the constraint and the quantity to be optimized.

240 = 2L + 2W \\ \\ \phantom{aaa} A = LW

We are interested in optimizing the second equation. There is a slight problem: it contains three unknown variables. Fortunately, these problems are designed so that you can reduce the equation to two variables. How do we do this? First, we isolate for one of the variables in the constraint equation. Let's isolate for L:

240 = 2L + 2W \\ \\ \phantom{aaa} 120 = L + W \\ \\ \phantom{aaa} L = 120 - W

Now we can substitute this expression into the equation that is to be optimized (in this case, the area equation).

A = LW \\ \\ \phantom{aaa} A = (120-W)W \\ \\ \phantom{aaa} A = -W^{2} + 120W

The area is now exclusively a function of the width and is a quadratic function. If we were to plot this function with W along the horizontal axis and A along the vertical axis, we would find something like this:

From this it is obvious that the maximum area will occur at precisely the vertex. Furthermore, we know that if we were to draw a tangent line at the vertex, it would be horizontal. In calculus, this means that the derivative at the location of the vertex is equal to zero. So we compute the derivative of the area function with respect to W, set it equal to zero and solve for W.

A = -W^{2} + 120W \\ \\ \phantom{aaa} A ' = -2W + 120 \\ \\ \phantom{aaa} 0 = -2W + 120 \\ \\ \phantom{aaa} W = 60

Finally, use this value of W to find L from the constraint equation.

240 = 2L + 2W \\ \\ \phantom{aaa} 240 = 2L + 120 \\ \\ \phantom{aaa} L = 60

The dimensions of the rectangle that maximize the area are 60 cm x 60 cm. Indeed, this is a square, which verifies the concepts we were taught in elementary school: a square always gives the maximum area of a four sided polygon with a fixed perimeter, only now we have used calculus to prove it. The maximum area is 3600 square centimeters.

Generalized Approach

Use the following algorithm to solve optimization problems:

  1. Determine what constraint(s) you are given and what quantity you are asked to optimize. Draw a diagram if you think it will help
  2. State the variables you will use and what each one represents
  3. Write out a constraint equation (or equations if more than one is needed) and an equation that needs to be optimized
  4. Isolate for one of the variables in the constraint equation so that you can substitute it into the equation that needs to be optimized
  5. Differentiate the equation to be optimized
  6. Set the differentiated equation equal to zero and solve for the independent variable
  7. Determine any other information you would need to answer the problem

Keep in mind that this algorithm is just a general guideline to solving optimization problems. Sometimes you may require different strategies.
