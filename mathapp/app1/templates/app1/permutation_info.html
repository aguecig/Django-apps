Fundamental Principle of Counting

To determine how many ways we can arrange a collection of A number of objects with B number of objects, we do this by multiplying A and B. For example, in how many ways can we select a cube from a set of 6 and a cone from a set of 3? Well, there are 6 ways to select the cube and 3 ways to select the cone, so in total there are (6)(3) = 18 ways to select a cube and a cone.

How many ways can we arrange n objects? Well, there are n ways to select the first object, n-1 ways to select the second object, n-2 ways to select the third object, and so on. Using the fundamental counting principle, the total number of ways to do this is given by

(n)(n-1)(n-2)...(3)(2)(1) \\ \\ \phantom{aaa} = n!

This is called a factorial, and is read as "n factorial". Factorials are computed by simply multiplying all natural numbers from 1 to n. So, for example, 6! = (1)(2)(3)(4)(5)(6) = 720. You can find the ! button on your calculator to quickly compute factorials.

Permutations

What if we don't want to arrange all objects from a group of n, but only some of them? This can be calculated by using a permutation. Suppose we have n objects and we wish to determine how many ways we can arrange r of them, where r <= n. This is done using the following formula

\frac{n!}{(n-r)!} \\ \\ \phantom{aaa} = \ _{n}P_{r}

Suppose we 10 coins and we wish to know how many ways we can order 5 of them. We can use the formula, or the permutation button on our calculator to find that

\ _{10}P_{5} = 30 240

Key words in word problems

When you are given a word problem that involves counting, it is important to pay attention to the wording of the question. Take the following two similar, but computationally different, questions:

  1. Given a group of 5 boys and 4 girls, in how many ways can you select 2 boys and 3 girls?
  2. Given a group of 5 boys and 4 girls, in how many ways can you select 2 boys or 3 girls?

In this first case, the boys and girls to be selected are part of the same group, so we should just multiply the number of ways to select 2 boys and 3 girls:

(_{5}P_{2})(_{4}P_{3}) = 480

In the second case, the two situations happen seperately, so we should calculate the number of ways to select 2 boys and add it to the number of ways to select 3 girls:

_{5}P_{2} \ + \ _{4}P_{3} = 44

Remember that and is multiplication and or is addition.
