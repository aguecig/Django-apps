Remainder Theorem and Factor Theorem

Given a polynomial function f(x) and a binomial x - r, the remainder theorem states that when f(x) is divided by x - r, the remainder is f(r).

Given a polynomial function f(x) and a binomial x - r, the factor theorem states that x - r is a factor of f(x) if and only if f(r) = 0.

Even and Odd functions

The condition for a function f(x) to be classified as an even function is given by:

f(-x) = f(x)

The condition for a function f(x) to be classified as an odd function is given by:

f(-x) = -f(x)

If neither of these conditions are meant, the function is said to be neither even nor odd.
