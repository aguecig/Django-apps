Analyze Quadratics

Forms of Quadratic Functions

Standard Form: c is the y-intercept

f(x) = ax^{2} + bx + c

Factored Form: s and t are the x-intercepts

f(x) = a(x-s)(x-t)

Vertex Form: (h,k) is the coordinates of the vertex

f(x) = a(x-h)^{2} + k

In each case, a represents the direction of opening. If a is positive, the parabola will open up. If a is negative, the parabola will open down.

Quadratic Formula and the Discriminant

The quadratic formula can be used to solve an equation in the following form:

ax^{2} + bx + c = 0

In which case the solutions are given by the quadratic formula

x = \frac{-b \pm \sqrt{b^{2} - 4ac}}{2a}

It is possible to know how many real roots there are by evaluating the discriminant, which is given by

b^{2}-4ac

If the discriminant is less than 0, there are no real roots. If it is equal to 0, there is exactly one real root. Finally, if the discriminant is larger than 0, there are exactly two real roots.

Transformations of Quadratic Functions

The general formula for a quadratic transformation is given by

f(x) = a[c(x-h)]^{2} + k
  • a
    • If negative, a reflection in the x-axis
    • If larger than 1, a vertical stretch
    • If between 0 and 1, a vertical compression
  • c
    • If negative, a reflection in the y-axis
    • If larger than 1, a horizontal compression
    • If between 0 and 1, a horizontal stretch
  • h
    • If negative, a translation left
    • If positive, a translation right
  • k
    • If negative, a translation down
    • If positive, a translation up

Guidelines for Solving a Quadratic Equation

Take the following example

x(x+2) = 2x + 5 + 2(x-1)

Expand and simplify as needed

x^{2} + 2x = 2x + 5 + 2x - 2 \\ \phantom{aaa} x^{2} + 2x = 4x + 3

Collect all terms on one side of the equation to have it all equal to 0. When doing this, you want the squared term to have a positive coefficient.

x^{2} - 2x - 3 = 0

Factor if possible, or else use the quadratic formula to solve.

(x-3)(x+1) = 0

Finally, write out the solution(s)

x = -1, \ \ x = 3
