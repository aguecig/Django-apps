Quadratic Formula

Use when trying to solve equation of the form

ax^{2}+bx+c=0

Then the solutions are given by

x = \frac{-b \pm \sqrt{b^{2}-4ac}}{2a}

The discriminant is

b^{2}-4ac

And can be used to determine the number of real roots:

  • No real roots if less than 0
  • Exactly one real root if equal to 0
  • Exactly two real roots if greater than 0
