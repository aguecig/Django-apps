How to analyze a rational function

Using the example:

f(x) = \frac{x^{2}-2x-24}{x^{2}-36}
  • Factor the numerator and denominator if possible
    f(x) = \frac{(x-6)(x+4)}{(x-6)(x+6)}
  • Simplify and determine the location of any holes
    f(x) = \frac{x+4}{x+6}

    Hole at (6, 5/6)

    • Use the simplified function for the rest of the algorithm

  • Use the denominator to find the location of any vertical asymptotes
    x = -6
  • Determine if there are any horizontal or vertical asymptotes
    • If deg(numerator) is 1 more than deg(denominator), there is an oblique asymptote
    • If deg(numerator) is less than deg(denominator), horizontal asymptote is y = 0
    • If deg(numerator) = deg(denominator), divide lead coefficients to find horizontal asymptote:
      • y = \frac{1}{1} = 1
  • Use a chart to find the behaviour of the function as it approaches either side of all asymptotes
    x \rightarrow -6^{-} \quad f(x) \rightarrow \infty \\ \phantom{aa}x \rightarrow -6^{+} \quad f(x) \rightarrow -\infty \\ \\ \phantom{aa}x \rightarrow \infty \quad \quad f(x) \rightarrow 1^{-} \\ \phantom{aa}x \rightarrow -\infty \quad f(x) \rightarrow 1^{+} \\
  • Use the numerator to locate any x-intercepts
    x_{int} = -4
  • Sub in x = 0 to locate the y-intercept
    y_{int} = \frac{2}{3}
  • Graph the function
