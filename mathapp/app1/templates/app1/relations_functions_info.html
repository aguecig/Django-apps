Definitions

A relation is a relationship between sets of values. In math, the relation is between the x-values and y-values of ordered pairs. The set of all x-values is called the domain, and the set of all y-values is called the range.


A function f from a set of numbers in a domain D to a set of numbers in a range R is defined such that every element of the domain D corresponds to exactly one element in the range R.

How to determine if a relation is a function

If we are given a table of values, so long as no x-values repeat themselves, the table will represent a function

If we are given an algebraic equation, then we must determine if the graphical representation of the relation would pass the vertical line test.

The vertical line test works as follows: if you can pass a vertical line through every point of the graph of a relation and the line never crosses the graph in more than one location, then the graph represents a function.
