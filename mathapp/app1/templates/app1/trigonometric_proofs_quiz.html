Info Sheet

In this section, you will find a variety of questions that are statements of trigonometric identities. Your goal is to prove that each identity is true. If you need help, you can either go to the forums and try and post your solution there, or you can click the Show Solution button at the bottom of each question. Note however, that the solution shown is only one possible way to prove the identity. For example, you may wish to prove the identity using the other side of the equation.


Question 1:

\tan{x}\sin{x}+\cos{x}=\sec{x}
\tan{x}\sin{x}+\cos{x} \\ \\ = \frac{ \sin{x} }{ \cos{x} }\sin{x}+\cos{x} \\ \\ = \frac{ \sin{^{2}x} }{ \cos{x} } + \frac{ \cos{x} }{1} \\ \\ = \frac{ \sin{^{2}x} }{ \cos{x} } + \frac{ \cos{^{2}x} }{\cos{x}} \\ \\ = \frac{ \sin{^{2}x} + \cos{^{2}x} }{\cos{x}} \\ \\ = \frac{ 1 }{ \cos{x}} \\ \\ = \sec{x} \\ \\

Question 2:

\tan{x} + \cot{x} = \sec{x}\csc{x}
\tan{x} + \cot{x} \\ \\ =\frac{ \sin{x} }{ \cos{x} } + \frac{ \cos{x} }{ \sin{x} } \\ \\ =\frac{ \sin{^{2}x} }{ \cos{x}\sin{x} } + \frac{ \cos{^{2}x} }{ \cos{x}\sin{x} } \\ \\ =\frac{ \sin{^{2}x} + \cos{^{2}x} }{ \cos{x}\sin{x} } \\ \\ =\frac{ 1 }{ \cos{x}\sin{x} } \\ \\ =\frac{ 1 }{ \cos{x} } \frac{ 1 }{ \sin{x} } \\ \\ = \sec{x}\csc{x} \\ \\

Question 3:

\sin{x} - \sin{x}\cos{^{2}x} = \sin{^{3}x}
\sin{x} - \sin{x}\cos{^{2}x} \\ \\ = \sin{x} (1 - \cos{^{2}x}) \\ \\ = \sin{x} \sin{^{2}x} \\ \\ = \sin{^{3}x} \\

Question 4:

\cos{^{2}x} = \frac{ \csc{x}\cos{x} }{ \tan{x} + \cot{x} }
\frac{ \csc{x}\cos{x} }{ \tan{x} + \cot{x} } \\ \\ \\ = \frac{ \frac{ 1 }{ \sin{x} } \frac{ \cos{x} }{ 1 } }{ \frac{ \sin{x} }{ \cos{x} } + \frac{ \cos{x} }{ \sin{x} } } \\ \\ \\ = \frac{ \frac{ \cos{x} }{ \sin{x} } }{ \frac{ \sin{^{2}x} }{ \cos{x}\sin{x} } + \frac{ \cos{^{2}x} }{ \cos{x}\sin{x} } } \\ \\ \\ = \frac{ \frac{ \cos{x} }{ \sin{x} } }{ \frac{ \sin{^{2}x} + \cos{^{2}x} }{ \cos{x}\sin{x} } } \\ \\ \\ = \frac{ \frac{ \cos{x} }{ \sin{x} } }{ \frac{ 1 }{ \cos{x}\sin{x} } } \\ \\ \\ = \frac{ \cos{x} }{ \sin{x} } \frac{ \cos{x}\sin{x} }{ 1 } \\ \\ \\ = \cos{^{2}x}

Question 5:

\frac{ 1 - \sin{x} }{ \cos{x} } = \frac{ \cos{x} }{ 1 + \sin{x} }
\frac{ 1 - \sin{x} }{ \cos{x} } \\ \\ \\ = \frac{ 1 - \sin{x} }{ \cos{x} } \frac{ 1 + \sin{x} }{ 1 + \sin{x} } \\ \\ \\ = \frac{ (1 - \sin{x})(1 + \sin{x}) }{ \cos{x}(1 + \sin{x}) } \\ \\ \\ = \frac{ 1 - \sin{x} + \sin{x} - \sin{^{2}x} }{ \cos{x}(1 + \sin{x}) } \\ \\ \\ = \frac{ 1 - \sin{^{2}x} }{ \cos{x}(1 + \sin{x}) } \\ \\ \\ = \frac{ \cos{^{2}x} }{ \cos{x}(1 + \sin{x}) } \\ \\ \\ = \frac{ \cos{x} }{ 1 + \sin{x} } \\ \\ \\

Question 6:

\frac{ \sin{^{4}x} - \cos{^{4}x} }{ \sin{^{2}x} - \cos{^{2}x} } = 1
\frac{ \sin{^{4}x} - \cos{^{4}x} }{ \sin{^{2}x} - \cos{^{2}x} } \\ \\ = \frac{ (\sin{^{2}x} + \cos{^{2}x})(\sin{^{2}x} - \cos{^{2}x}) }{ \sin{^{2}x} - \cos{^{2}x} } \\ \\ = \frac{ (1)(\sin{^{2}x} - \cos{^{2}x}) }{ \sin{^{2}x} - \cos{^{2}x} } \\ \\ = 1 \\ \\

Question 7:

\frac{ \sec{x} - 1 }{ \sec{x} + 1 } = \frac{ 1 - \cos{x} }{ 1 + \cos{x} }
\frac{ 1 - \cos{x} }{ 1 + \cos{x} } \\ \\ \\ = \frac{ 1 - \cos{x} }{ 1 + \cos{x} } \frac{ \frac{ 1 }{ \cos{x} } }{ \frac{ 1 }{ \cos{x} } } \\ \\ \\ = \frac{ (1 - \cos{x})\frac{ 1 }{ \cos{x} } }{ (1 + \cos{x})\frac{ 1 }{ \cos{x} } } \\ \\ \\ = \frac{ \frac{ 1 }{ \cos{x} } - 1 }{ \frac{ 1 }{ \cos{x} } + 1 } \\ \\ \\ = \frac{ \sec{x} - 1 }{ \sec{x} + 1 } \\ \\ \\

Question 8:

\frac{ 1 + \sin{x} }{ 1 - \sin{x} } - \frac{ 1 - \sin{x} }{ 1 + \sin{x} } = 4\tan{x}\sec{x}
\frac{ 1 + \sin{x} }{ 1 - \sin{x} } - \frac{ 1 - \sin{x} }{ 1 + \sin{x} } \\ \\ \\ = \frac{ (1 + \sin{x})(1 + \sin{x}) }{ (1 - \sin{x})(1 + \sin{x}) } - \frac{ (1 - \sin{x})(1 - \sin{x}) }{ (1 + \sin{x})(1 - \sin{x}) } \\ \\ \\ = \frac{ 1 + 2\sin{x} + \sin{^{2}x} }{ (1 - \sin{x})(1 + \sin{x}) } - \frac{ 1 - 2\sin{x} + \sin{^{2}x} }{ (1 - \sin{x})(1 + \sin{x}) } \\ \\ \\ = \frac{ 4\sin{x} }{ 1 -\sin{x} + \sin{x} - \sin{^{2}x} } \\ \\ \\ = \frac{ 4\sin{x} }{ 1 - \sin{^{2}x} } \\ \\ \\ = \frac{ 4\sin{x} }{ \cos{^{2}x} } \\ \\ \\ = \frac{ 4\sin{x} }{ \cos{x}\cos{x} } \\ \\ \\ = \frac{ 4\sin{x} }{ \cos{x} }\frac{ 1 }{ \cos{x} } \\ \\ \\ = 4\tan{x}\sec{x} \\ \\ \\

Question 9:

\sin{x}(\cot{x} + \tan{x}) = \sec{x}
\sin{x}(\cot{x} + \tan{x}) \\ \\ \\ = \sin{x} \left( \frac{ \cos{x} }{ \sin{x} } + \frac{ \sin{x} }{ \cos{x} } \right) \\ \\ \\ = \cos{x} + \frac{ \sin{^{2}x} }{ \cos{x} } \\ \\ \\ = \frac{ \cos{^{2}x} }{ \cos{x} } + \frac{ \sin{^{2}x} }{ \cos{x} }\\ \\ \\ = \frac{ \cos{^{2}x} + \sin{^{2}x} }{ \cos{x} }\\ \\ \\ = \frac{ 1 }{ \cos{x} }\\ \\ \\ = \sec{x}
